# Job Tracker (Multi-Tenant)

A full-stack job tracking application with tenant-based data isolation.

## 🚀 Tech Stack
- Frontend: Next.js, React, Tailwind
- Backend: Node.js, Express
- DB: PostgreSQL (multi-tenant)
- ORM: Prisma

## ✨ Features
- Multi-tenant architecture (TCS, Wipro, Infosys)
- Job management
- Application tracking
- Status updates
- Pagination + filtering

## ⚙️ Setup

### Backend
cd backend
npm install
npx prisma migrate dev
npm run dev

### Frontend
cd frontend
npm install
npm run dev
